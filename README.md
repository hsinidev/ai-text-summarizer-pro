The **AI Text Summarizer** redefines how users interact with Large Language Models for text processing. Unlike standard tools that act as "black boxes," processing your data on opaque servers, this application acts as a secure **client-side interface**.

It connects directly from your browser to your specified LLM endpoint (Ollama, Cloudflare Workers AI, or OpenAI-compatible APIs). This ensures **zero data latency** and **100% data sovereignty**.

## ğŸš€ Usage Guide

1.  **Access the App:** Go to [doodax.com/ai-summarizer](https://doodax.com/ai-summarizer/).
2.  **Configure Endpoint:**
    *   Locate the "LLM Endpoint URL" field.
    *   Enter your API URL (e.g., `https://your-ollama-instance.com/api/generate`).
    *   *Note: Ensure your endpoint allows CORS.*
3.  **Summarize:**
    *   Paste your text.
    *   Adjust length (1-10 sentences).
    *   Click **Summarize Now**.
